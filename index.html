11.2 Shopify 后台操作指南

添加/编辑产品

  1. 登录 https://admin.shopify.com
  2. 左侧菜单点击 Products（产品）
  3. 点击 Add product（添加产品） 或选择已有产品编辑
  4. 填写产品信息（标题、描述、价格、库存、图片）
  5. 设置 SEO meta 标签（点击产品页面底部 Edit website SEO）
  6. 点击 Save（保存）

处理订单

  1. 登录 Shopify 后台
  2. 左侧菜单点击 Orders（订单）
  3. 查看新订单状态（New / Unfulfilled / Fulfilled）
  4. 点击订单号 → 查看详情 → 点击 Fulfill（发货）
  5. 打印运单或手动填写物流信息

查看销售数据

  1. 登录 Shopify 后台
  2. 左侧菜单点击 Analytics（分析）
  3. 查看 Dashboard（概览）、Orders（订单）、Sessions（访问量）
  4. 可按日/周/月筛选时间范围

11.3 Cloudflare Pages 部署管理

查看部署状态

  1. 登录 Cloudflare Dashboard
  2. 选择 Workers & Pages
  3. 找到 ouni/ounin 项目
  4. 查看 Deployment（部署）选项卡

手动触发重新部署

  1. 在 Cloudflare Pages 项目中
  2. 点击 Deployments（部署）
  3. 选择最近的 deployment 点击 Retry deployment

查看构建日志

  1. 点击某个 deployment
  2. 选择 View logs（查看日志）
  3. 查看构建过程是否有错误
⚠️ 部署失败排查

如果页面没有更新，检查：
1. GitHub 仓库是否有最新代码
2. Cloudflare Pages 是否有部署记录
3. 构建日志是否有报错（常见：文件路径错误、HTML语法错误）

11.6 DNS 配置管理（Cloudflare）

添加新的 DNS 记录

  1. 登录 Cloudflare Dashboard
  2. 选择域名 ounsshop.com
  3. 点击 DNS → Add record（添加记录）
  4. 选择类型（A / CNAME / TXT 等）
  5. 填写名称和内容 → 保存

修改 SSL/TLS 设置

  1. 登录 Cloudflare Dashboard
  2. 选择域名 → SSL/TLS
  3. 选择加密模式（Full 推荐）
💡 常用 DNS 记录类型
  • A 记录：将域名指向一个 IPv4 地址
  • CNAME：将域名指向另一个域名（如 store 指向 Shopify）
  • TXT：用于验证域名所有权（如 Google Search Console）
